## 统计口径

- 类别图像数：一张图片只要包含某类别的至少一个有效实例，该类别的图像数增加 1；同一张图内重复出现不重复计算。
- 类别实例数：标签文件中每一行类别 ID 合法的标注算作一个实例；多边形坐标异常会另外记录，不从类别数量中扣除。
- 空标签：标签文件存在但没有任何非空标注行，视为无缺陷背景样本，不自动当作错误。
- 缺失标签：图片存在，但找不到同相对路径、同基本文件名的 `.txt` 文件，需要单独确认。
- 孤立标签：标签存在，但找不到对应图片，需要单独确认。

后续的小、中、大目标划分标准将在完成图像尺寸和目标相对面积统计后确定，并在数据报告中记录。
